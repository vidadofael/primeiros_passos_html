Principais formatações

Negrito / Destaques

Nesta frase tem termo em negrito usando a tag <b> (não semântico)

Nesta frase tem um termo em destaque usando a tag <strong> (semântico)

Itálico / Ênfase

Nesta frase tem um termo em destaque usando a tag <i> (não semântica)

Nesta frase tem um termo em destaque usando a tag <em> (semântica)

Podemos também criar "marca texto" usando a tag <mark>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. Tag <del>

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado. Tag <ins>.

Podemos sublinhar um texto também usando a tag <u>.

Texto subescrito

Para inserir coisa do tipo x20 + 3 com a tag <sup>.

Texto sbscristo

Para inserir coisaa do tipo H2O com a tag <sub>.
