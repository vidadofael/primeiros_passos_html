Usando links externos

Você pode acessar o meu repositório público no GitHub.

Usa-se as tags para link <a> e dentro dela para abrir uma nova página usa se os atributos dentro da tag a assim:

<a href="https://github.com/vidadofael" target="_blanck" rel="external">repositório público no GitHub.</a></p>

Desta forma o usuário não perde a sua página original, pois clicando no link abrirá uma nova aba fazendo com que a sua página não feche com a abertura de uma nova página.


Agora neste exemplo não tem os atributos dentro do link que faz com que o link abra na mesma página fazendo com que o usuário perca a página do seu site dependendo da usabilidade.

Você também pode acessaar meu perfil no Youtube.
